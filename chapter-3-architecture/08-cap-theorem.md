## CAP theorem

CAP theorem, or Brewer's theorem, named after the computer scientist Eric Brewer, defines the trilemma on distributed data storage. The theory suggests that in a distributed data storage, of the three properties — Consistency, Availability and Partition tolerance — we can only have two out of the three. As we are building a distributed system, this theorem is worthwhile to understand. As with all engineering, it's about a set of trade-offs.

While this is somewhat theoretical, in a real world scenario the properties are not binary, but rather levels of.
